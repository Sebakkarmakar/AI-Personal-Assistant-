# AI Personal Assistant

An AI-powered web application built with **Flask** and **Google Gemini API** that can answer user questions and summarize emails. The application provides a simple and interactive interface for users to communicate with an AI assistant.

## Features

* 💬 Ask AI any question and receive intelligent responses.
* 📧 Summarize long emails into concise, easy-to-read summaries.
* ⚡ Fast response generation using Google Gemini 2.5 Flash.
* 🌐 Simple and responsive web interface built with HTML, CSS, and JavaScript.

## Technologies Used

* Python
* Flask
* Google Gemini API (`google-genai`)
* HTML
* CSS
* JavaScript
* Python Dotenv

## Usage

### Ask AI

* Enter your question.
* Click the **Ask** button.
* The AI generates an answer instantly.

### Email Summarizer

* Paste an email into the text area.
* Click **Summarize**.
* The AI returns:

  * Main purpose
  * Important points
  * Action items
  * Deadlines (if any)
